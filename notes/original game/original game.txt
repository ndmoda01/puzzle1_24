modes
Krazy Kreatures mode (Classic)
- Specific number of unique board shapes (levels), currently 10
- Each level gets more difficult
- Blocks do not spawn in, but may be present at the start of the level
- New items spawn in continuously until the timer nearly runs out
- Last 10 seconds, no new items spawn in. Bonus given for cleared sections of the board

Single player 
	Single Player: 1 player goes through all levels to try to get to the end with the highest score
		Win: complete all levels and end with highest score possible
		Lose: if board fills up completely
	
Multi-player - 
	Coop (shared board): multiple players go through all levels to try to get to the end with the highest score
		Win: complete all levels with highest score possible (player scores combine)
		Lose: if board fills up completely, both players lose
		
	Comp (split board): multiple players go through all levels to try to get to the end with the highest score
		Win: complete all levels with highest score possible (or higher than your opponent) 
		Lose: if a player's board fills up completely, they are knocked out of the round. If both fill up in the same round, game over


Tetris mode (Endurance)
Single player 
	Single Player: 1 player progresses as far as possible on the same board. Each level gradually gets more difficult
		Win: complete as many stages as possible with highest score possible (no discrete end?)
		Lose: if board fills up completely
	
Multi-player - 
	Coop (shared board): multiple players progress as far as possible on 1 board. Each level gradually gets more difficult
		Win: complete as many stages as possible with highest score possible (no discrete end?)
		Lose: if board fills up completely, both players lose
		
	Comp (split board): multiple players progress as far as possible on separate boards. Each level gradually gets more difficult
		Win: complete as many stages as possible with a higher score than all other players (no discrete end?)
		Lose: if a player's board fills up completely, they are knocked out (lose a life). Once a player loses all thier lives and is knocked out, they are out
